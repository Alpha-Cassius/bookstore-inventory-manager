# Bookstore Inventory Manager 📚

A simple Python-based tool to manage bookstore inventory using JSON for persistent data storage. This project demonstrates how to read, update, and format data using Python's built-in `json` and file-handling libraries.

## 🚀 Features

* **Data Persistence**: Uses `inventory.json` to save book details (Title, Author, Price, and Stock).
* **Automatic Updates**: Appends new books to the existing database without overwriting previous entries.
* **Formatted Output**: Displays the inventory in a clean, human-readable format.

## 📂 File Structure

* `main.py`: The core logic for file handling and data manipulation.
* `inventory.json`: The data file where the bookstore inventory is stored.
